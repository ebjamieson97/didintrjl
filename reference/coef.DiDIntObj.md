# Extract coefficients from `DiDIntObj`

Extract coefficients from `DiDIntObj`

## Usage

``` r
# S3 method for class 'DiDIntObj'
coef(object, level = c("agg", "sub"), ...)
```

## Arguments

- object:

  A `DiDIntObj` object

- level:

  Specify either `"agg"` or `"sub"` to view the aggregate or
  sub-aggregate results.

- ...:

  other arguments

## Value

A data frame of coefficient estimates
